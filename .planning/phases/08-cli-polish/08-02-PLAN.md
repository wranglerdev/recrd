    **Create `apps/recrd-cli/Ipc/SessionSocket.cs`** (per D-02, D-03):
    - `internal sealed class SessionSocket : IAsyncDisposable`
    - Constructor takes `string socketPath`, `ILogger logger`
    - Socket path: `Path.Combine(Environment.GetFolderPath(Environment.SpecialFolder.UserProfile), ".recrd", "session.sock")`
    - `public static string DefaultSocketPath` — the above path as a static property
    - `public static async Task<bool> IsSessionActiveAsync(string socketPath)` — checks if file exists, tries to connect. If connect fails with SocketException, delete stale file and return false. If connects, return true. If file doesn't exist, return false. (Per Pitfall 3 from RESEARCH.md)
    - `public async Task StartListeningAsync(Func<string, Task> commandHandler, CancellationToken ct)`:
      1. Ensure `~/.recrd/` directory exists (`Directory.CreateDirectory`)
      2. Create `UnixDomainSocketEndPoint(socketPath)`
      3. Create `Socket(AddressFamily.Unix, SocketType.Stream, ProtocolType.Unspecified)`
      4. Bind, Listen(1)
      5. Loop: `AcceptAsync(ct)` -> `NetworkStream` -> `StreamReader` -> `ReadLineAsync` -> deserialize `IpcMessage` -> call `commandHandler(msg.Command)` -> if command is "stop", break
      6. In finally block: `File.Delete(socketPath)`
    - `public async ValueTask DisposeAsync()` — delete socket file if it exists

    **Create `apps/recrd-cli/Ipc/SessionClient.cs`** (per D-02, D-04):
    - `internal static class SessionClient`
    - `public static async Task<int> SendCommandAsync(string command, ILogger logger, CancellationToken ct)`:
      1. Build socket path from `SessionSocket.DefaultSocketPath`
      2. If socket file doesn't exist: `CliOutput.WriteError("No active session found. Start one with 'recrd start'.")`, return 1
      3. Create `Socket(AddressFamily.Unix, SocketType.Stream, ProtocolType.Unspecified)`
      4. `ConnectAsync(new UnixDomainSocketEndPoint(socketPath), ct)`
      5. Create `NetworkStream` -> `StreamWriter { AutoFlush = true }`
      6. `WriteLineAsync(JsonSerializer.Serialize(new IpcMessage(command)))`
      7. Return 0

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-8-03 | Spoofing | Unix socket path (~/.recrd/session.sock) | accept | Single-user local tool; socket created in user home directory with default permissions. Malicious socket pre-creation requires local access already compromised. |
| T-8-04 | Tampering | IPC message content | mitigate | Validate deserialized IpcMessage.Command against known values ("pause", "resume", "stop"); ignore unknown commands with warning log |
| T-8-05 | Denial of Service | Stale session.sock blocks new sessions | mitigate | IsSessionActiveAsync probes socket; deletes stale files when no listener responds (Pitfall 3) |
</threat_model>
