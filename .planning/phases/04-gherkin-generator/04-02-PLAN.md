2. Create `packages/Recrd.Gherkin/Internal/StepTextRenderer.cs`:
   - `namespace Recrd.Gherkin.Internal;`
   - `internal static class StepTextRenderer`
   - Public method: `internal static string Render(IStep step)` — pattern-matches on step type:
     - `ActionStep a` => dispatch to `RenderAction(a)`
     - `AssertionStep a` => dispatch to `RenderAssertion(a)`
     - Otherwise: `throw new InvalidOperationException($"Unknown step type: {step.GetType().Name}")`

   - `private static string BestSelectorValue(Selector selector)`:
     - Walk preference order: `SelectorStrategy.DataTestId`, `SelectorStrategy.Id`, `SelectorStrategy.Role`
     - For each, check `selector.Values.TryGetValue(strategy, out var value)` — if found, return value
     - Fallback: iterate `selector.Strategies` and return first value found in `selector.Values`
     - Ultimate fallback: return `"(unknown)"`

   - `private static string RenderAction(ActionStep step)` using templates per D-01:
     - `Navigate` => `Navega para "{Payload["url"] ?? Payload.GetValueOrDefault("href") ?? BestSelectorValue(step.Selector)}"`
     - `Click` => `Clica no elemento "{BestSelectorValue(step.Selector)}"`
     - `Type` => `Digita "{Payload["value"]}" no campo "{BestSelectorValue(step.Selector)}"` — if Payload["value"] contains `<` and `>`, keep it as-is (variable placeholder per D-02)
     - `Select` => `Seleciona "{Payload["value"]}" no campo "{BestSelectorValue(step.Selector)}"`
     - `Upload` => `Envia o arquivo "{Payload.GetValueOrDefault("filename") ?? Payload.GetValueOrDefault("path") ?? "(unknown)"}" no campo "{BestSelectorValue(step.Selector)}"`
     - `DragDrop` => `Arrasta "{BestSelectorValue(step.Selector)}" para "{Payload.GetValueOrDefault("target") ?? "(unknown)"}"`

   - `private static string RenderAssertion(AssertionStep step)`:
     - `TextEquals` => `O texto do elemento "{BestSelectorValue(step.Selector)}" e' "{Payload["expected"]}"` (use proper accent: `\u00e9`)
     - `TextContains` => `O texto do elemento "{BestSelectorValue(step.Selector)}" cont\u00e9m "{Payload["expected"]}"`
     - `Visible` => `O elemento "{BestSelectorValue(step.Selector)}" est\u00e1 vis\u00edvel`
     - `Enabled` => `O elemento "{BestSelectorValue(step.Selector)}" est\u00e1 habilitado`
     - `UrlMatches` => `A URL corresponde a "{Payload["pattern"]}"`

3. Create `packages/Recrd.Gherkin/Internal/GroupingClassifier.cs`:
   - `namespace Recrd.Gherkin.Internal;`
   - `internal static class GroupingClassifier`
   - Method: `internal static IReadOnlyList<(GroupType Group, IStep Step)> Classify(IReadOnlyList<IStep> steps)`
   - Algorithm (two-pass default heuristic per GHER-06):
     - Find the index of the first `ActionStep` with `ActionType.Navigate` => `firstNavIdx`
     - If no Navigate found, `firstNavIdx = -1`
     - For each step at index `i`:
       - If step is `AssertionStep` => `GroupType.Then`
       - Else if `firstNavIdx == -1` => `GroupType.When` (no navigate, default all actions to When)
       - Else if `i <= firstNavIdx` => `GroupType.Given` (up to and including first navigate)
       - Else => `GroupType.When` (after first navigate)
     - Return list of `(GroupType, IStep)` tuples

Replace the stub `GherkinGenerator.cs` with the full implementation for the Cenario (non-data-driven) path. The data-driven path (Esquema do Cenario + Exemplos) will be added in Plan 03.

Implementation of `GenerateAsync`:

1. **Header** (per D-05, GHER-08):
   ```
   await output.WriteLineAsync("# language: pt");
   await output.WriteLineAsync();
   ```

2. **Feature line** (per D-03):
   ```
   var featureName = session.Metadata.BaseUrl ?? "(sem URL base)";
   await output.WriteLineAsync($"Funcionalidade: {featureName}");
   await output.WriteLineAsync();
   ```

3. **Tags** (per D-04):
   If `options?.Tags` is not null and not empty, write each tag as `@tag` before the scenario keyword, one per line, indented with 2 spaces.

4. **Scenario keyword branching** (GHER-01 vs GHER-02):
   - If `session.Variables.Count == 0` => write `  Cen\u00e1rio: <featureName>` (2-space indent)
   - If `session.Variables.Count > 0` => for now, write `  Esquema do Cen\u00e1rio: <featureName>` (Plan 03 handles the Exemplos table)

5. **Step emission**:
   Determine if steps are wrapped in GroupSteps or flat:
   - **GroupStep path** (GHER-05): If any top-level step is a `GroupStep`, iterate top-level steps. For each `GroupStep`:
     - Map `GroupType.Given` => `"Dado"`, `GroupType.When` => `"Quando"`, `GroupType.Then` => `"Ent\u00e3o"`
     - Emit first child step with primary keyword: `    {keyword} {StepTextRenderer.Render(child)}` (4-space indent)
     - Emit subsequent children with `    E {StepTextRenderer.Render(child)}`
     - For non-GroupStep top-level steps mixed in (edge case): render with `E` continuation

   - **Default heuristic path** (GHER-06): If NO top-level step is a `GroupStep`, call `GroupingClassifier.Classify(session.Steps)`:
     - Track `currentGroup` (initially null)
     - For each `(group, step)` in classified list:
       - If `group != currentGroup` => this is the first step in a new region, use primary keyword (`Dado`/`Quando`/`Ent\u00e3o`)
       - Else => use `E` continuation
       - Write: `    {keyword} {StepTextRenderer.Render(step)}`
       - Update `currentGroup = group`

6. **Trailing newline**: Ensure file ends with a single newline after the last step.

7. **Flush**: `await output.FlushAsync();`

Important: Do NOT call `DateTime.Now`, `Guid.NewGuid()`, or any non-deterministic operation anywhere in the emission path (GHER-07). All values come from the Session AST and Payload dictionaries which are already strings.
