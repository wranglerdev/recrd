Replace the stub `JsonDataProvider.cs` with full implementation per D-03, D-04, D-05, D-07.

**Constructor:** `JsonDataProvider(string filePath)` storing the field. Validate non-null.

**StreamAsync implementation using System.Text.Json:**

1. Open file stream: `await using var stream = File.OpenRead(_filePath)`.

2. Call `JsonSerializer.DeserializeAsyncEnumerable<JsonElement>(stream, cancellationToken: cancellationToken)` to get the async enumerable.

3. **Critical: non-array root detection.** Per research pitfall #3, `DeserializeAsyncEnumerable` does NOT throw immediately on a non-array root. The `JsonException` fires during the first `MoveNextAsync()`. Therefore wrap the `await foreach` in try/catch for `JsonException` and translate:
   ```csharp
   try
   {
       await foreach (var element in JsonSerializer.DeserializeAsyncEnumerable<JsonElement>(
           stream, cancellationToken: cancellationToken))
       {
           var flat = FlattenElement(element, string.Empty);
           yield return flat;
       }
   }
   catch (JsonException ex)
   {
       throw new DataParseException(
           lineNumber: (int)(ex.LineNumber ?? 0) + 1,
           offendingLine: string.Empty,
           filePath: _filePath,
           message: $"JSON root must be an array. {ex.Message}",
           innerException: ex);
   }
   ```

4. **FlattenElement private static method** — recursive dot-notation flattening per D-03 and D-04:
   ```csharp
   private static Dictionary<string, string> FlattenElement(JsonElement element, string prefix)
   ```
   - If `element.ValueKind != JsonValueKind.Object`, return empty dictionary
   - Iterate `element.EnumerateObject()`. For each property:
     - Compute `key` = `string.IsNullOrEmpty(prefix) ? prop.Name : $"{prefix}.{prop.Name}"`
     - `JsonValueKind.Object` -> recurse: `FlattenElement(prop.Value, key)` and merge all entries into result
     - `JsonValueKind.Array` -> **silently skip** per D-04 (do nothing)
     - `JsonValueKind.Null` -> `result[key] = ""` (null becomes empty string)
     - `JsonValueKind.True` -> `result[key] = "True"`
     - `JsonValueKind.False` -> `result[key] = "False"`
     - All other kinds (`String`, `Number`) -> `result[key] = prop.Value.ToString()`

**Required usings:**
```csharp
using System.Runtime.CompilerServices;
using System.Text.Json;
using Recrd.Core.Interfaces;
```

**Important:** Do NOT call `.ToList()` or buffer elements. The `yield return` inside `await foreach` maintains streaming.
