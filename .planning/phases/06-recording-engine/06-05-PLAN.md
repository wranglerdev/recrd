1. **Add popup tracking fields** to `PlaywrightRecorderEngine`:
```csharp
private readonly ConcurrentDictionary<IPage, string> _activePopups = new();
```

2. **Register `BrowserContext.Page` event handler** in `StartAsync`, after creating the recording context (per RESEARCH.md Pattern 5):
```csharp
_recordingContext.Page += (_, newPage) =>
{
    // ExposeFunctionAsync + AddInitScriptAsync are context-level — no action needed.
    // Just tag subsequent events from this page as popup-scoped.
    var popupId = Guid.NewGuid().ToString("N")[..8];
    _activePopups[newPage] = popupId;

    newPage.Close += (_, _) => _activePopups.TryRemove(newPage, out _);
};
```

3. **Update `HandleCapturedEvent`** to detect popup scope. The challenge: `ExposeFunctionAsync` callback does not tell us which page the event came from. We need the JS agent to include `window.location.href` in the event payload. Then in C#, we can check if the URL matches any tracked popup page.

   **Alternative approach (simpler):** Modify the JS recording agent to include a flag `isPopup: window.opener !== null` in every event payload. A page opened via `window.open` has `window.opener` set. This is simpler and more reliable than URL matching.

   Update `packages/Recrd.Recording/Scripts/recording-agent.js` — in the event dispatch function, add to the JSON payload:
   ```javascript
   isPopup: window.opener !== null
   ```

   Update `RecordedEventBuilder.Build` to read `isPopup` from the payload and include it in the `Payload` dictionary as `"__popupScope": "true"` if the event came from a popup.

4. **In `HandleCapturedEvent`**, after building the `RecordedEvent`:
```csharp
// The popup scope marker is already in evt.Payload["__popupScope"] from RecordedEventBuilder
// No additional handling needed — the event flows to the channel with the marker
```

5. **Ensure popup pages are closed on engine stop.** In `StopAsync`, before closing the recording context:
```csharp
foreach (var (page, _) in _activePopups)
{
    try { await page.CloseAsync(); } catch { /* already closed */ }
}
_activePopups.Clear();
```
