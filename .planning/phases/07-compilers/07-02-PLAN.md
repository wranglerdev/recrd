`Build(ActionType actionType, string selectorValue)`:
1. Look up verb from ActionType map
2. If `selectorValue` is empty/null (e.g. Navigate), return just the verb
3. Normalize slug: replace `-`, `_`, `.` with space; remove non-alphanumeric/non-space/non-accented chars; title-case each word (use `CultureInfo.InvariantCulture.TextInfo.ToTitleCase`); trim to 64 chars total
4. Return `"{verb} {slug}"`

`BuildAssertion(AssertionType assertionType, string selectorValue)`: Same pattern with AssertionType verb map.

**SelectorResolver.cs** — Replace stub with full implementation:

Define fallback chain: `SelectorStrategy[] FallbackChain = [DataTestId, Id, Role, Css, XPath]`

Browser format map (`FormatBrowser(SelectorStrategy strategy, string value)`):
- DataTestId -> `css=[data-testid="{value}"]`
- Id -> `id={value}`
- Role -> `role={value}`
- Css -> `css={value}`
- XPath -> `xpath={value}`

Selenium format map (`FormatSelenium(SelectorStrategy strategy, string value)`):
- DataTestId -> `css:[data-testid="{value}"]`
- Id -> `id:{value}`
- Role -> `css:[role="{value}"]`
- Css -> `css:{value}`
- XPath -> `xpath:{value}`

`ResolveBrowser(Selector selector, SelectorStrategy preferred)`:
1. Try preferred strategy first in selector.Values
2. Then walk FallbackChain in order, try each in selector.Values
3. If found, return `(FormatBrowser(strategy, value), warned: false)`
4. If fully exhausted, try `selector.Strategies.FirstOrDefault()` as last resort
5. If still nothing, return `("(unknown)", warned: true)`

`ResolveSelenium(Selector selector, SelectorStrategy preferred)`: Same logic using FormatSelenium.

**HeaderEmitter.cs** — Replace stub with full implementation:

`Emit(string targetName, string? sourceFilePath, Session? session)`:
1. Get version: `Assembly.GetExecutingAssembly().GetName().Version?.ToString(3) ?? "0.0.0"`
2. Get timestamp: `DateTimeOffset.UtcNow.ToString("o")`
3. Compute SHA-256: if `sourceFilePath` is not null and file exists, hash the file bytes; otherwise serialize `session` to JSON bytes via `JsonSerializer.Serialize(session, RecrdJsonContext.Default.Session)` and hash those bytes. Use `SHA256.HashData(bytes)` and convert to lowercase hex string.
4. Return two-line header string:
```
# Generated by recrd {version} at {timestamp} for target {targetName}
# Source: {sourceFilePath ?? "in-memory"} SHA-256: {sha256hex}
```
Use `\n` line endings (not `Environment.NewLine`).

```
namespace Recrd.Compilers.Internal;
internal static class BrowserKeywordEmitter
```

Uses 4-space separator constant: `private const string Sep = "    ";` (4 spaces — RF cell separator).

`WriteKeywordAsync(TextWriter writer, ActionStep step, CompilerOptions options, List<string> warnings)`:
For each ActionType, emit keyword body lines per RESEARCH.md mapping:

- **Click**: `Wait For Elements State{Sep}{selector}{Sep}visible{Sep}timeout=${TIMEOUT}s` then `Click{Sep}{selector}`
- **Type**: `Wait For Elements State{Sep}{selector}{Sep}visible{Sep}timeout=${TIMEOUT}s` then `Type Text{Sep}{selector}{Sep}{value}` where value = `step.Payload["value"]`
- **Select**: `Wait For Elements State...` then `Select Options By{Sep}{selector}{Sep}value{Sep}{value}`
- **Navigate**: `Go To{Sep}{url}` where url = `step.Payload.GetValueOrDefault("url") ?? step.Payload.GetValueOrDefault("href")`. NO Wait For Elements State (per Pitfall 7).
- **Upload**: `Wait For Elements State...` then `Upload File By Selector{Sep}{selector}{Sep}{path}` where path = `step.Payload.GetValueOrDefault("path") ?? step.Payload.GetValueOrDefault("filename")`
- **DragDrop**: `Wait For Elements State{Sep}{src}{Sep}visible{Sep}timeout=${TIMEOUT}s` then `Drag And Drop{Sep}{src}{Sep}{target}` where target = resolve second selector from `step.Payload["target"]`

Selector resolution uses `SelectorResolver.ResolveBrowser(step.Selector, options.PreferredSelectorStrategy)`. If `warned=true`, add warning string to the warnings list.

`WriteAssertionKeywordAsync(TextWriter writer, AssertionStep step, CompilerOptions options, List<string> warnings)`:
Per RESEARCH.md Browser AssertionType mapping:
- TextEquals: `Get Text{Sep}{selector}{Sep}=={Sep}{expected}`
- TextContains: `Get Text{Sep}{selector}{Sep}contains{Sep}{expected}`
- Visible: `Wait For Elements State{Sep}{selector}{Sep}visible`
- Enabled: `Wait For Elements State{Sep}{selector}{Sep}enabled`
- UrlMatches: `Get Url{Sep}=={Sep}{pattern}` (no selector — per Pitfall 7)

Where `expected`/`pattern` comes from `step.Payload["expected"]` or `step.Payload["pattern"]`.

```
*** Variables ***

*** Test Cases ***
Sessao Gravada
```

Then for each step: `    {keywordName}` (4-space indent + keyword name). For Type/Select steps with a value payload, append the value argument: `    Digitar Em Email Input    valor@exemplo.com`.
