<task type="auto">
  <name>Task 2: File Watcher and Auto-Update</name>
  <files>apps/vscode-extension/src/extension.ts</files>
  <action>
    Integrate file watching.
    - Use vscode.workspace.createFileSystemWatcher('**/*.recrd').
    - When a .recrd file is saved/changed:
        1. Run 'recrd compile' in the background for both 'gherkin' and 'robot-browser' targets.
        2. Read the resulting output files.
        3. Send a message to the PreviewPanel webview to refresh with new content.
    - Ensure it doesn't loop infinitely if compilation also modifies files (watch only .recrd).
  </action>
  <verify>
    <automated>grep "createFileSystemWatcher" apps/vscode-extension/src/extension.ts</automated>
  </verify>
  <done>Saving a recording session automatically updates the live preview.</done>
</task>
