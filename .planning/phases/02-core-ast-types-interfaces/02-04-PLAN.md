**RecrdJsonContext.cs** -- Source-generated context per D-06, D-07, D-08. CRITICAL: Must use `GenerationMode = JsonSourceGenerationMode.Metadata` for polymorphism support (see Research Pitfall 1).

```csharp
using System.Text.Json;
using System.Text.Json.Serialization;
using Recrd.Core.Ast;
using Recrd.Core.Interfaces;
using Recrd.Core.Pipeline;

namespace Recrd.Core.Serialization;

[JsonSourceGenerationOptions(
    PropertyNamingPolicy = JsonKnownNamingPolicy.CamelCase,
    GenerationMode = JsonSourceGenerationMode.Metadata,
    WriteIndented = false,
    DefaultIgnoreCondition = JsonIgnoreCondition.WhenWritingNull)]
[JsonSerializable(typeof(Session))]
[JsonSerializable(typeof(SessionMetadata))]
[JsonSerializable(typeof(ViewportSize))]
[JsonSerializable(typeof(Variable))]
[JsonSerializable(typeof(IStep))]
[JsonSerializable(typeof(ActionStep))]
[JsonSerializable(typeof(AssertionStep))]
[JsonSerializable(typeof(GroupStep))]
[JsonSerializable(typeof(ActionType))]
[JsonSerializable(typeof(AssertionType))]
[JsonSerializable(typeof(GroupType))]
[JsonSerializable(typeof(SelectorStrategy))]
[JsonSerializable(typeof(Selector))]
[JsonSerializable(typeof(RecordedEvent))]
[JsonSerializable(typeof(EventType))]
[JsonSerializable(typeof(CompilerOptions))]
[JsonSerializable(typeof(CompilationResult))]
[JsonSerializable(typeof(IReadOnlyList<IStep>))]
[JsonSerializable(typeof(IReadOnlyList<Variable>))]
[JsonSerializable(typeof(IReadOnlyList<Selector>))]
[JsonSerializable(typeof(IReadOnlyList<string>))]
[JsonSerializable(typeof(Dictionary<string, string>))]
[JsonSerializable(typeof(IReadOnlyDictionary<string, string>))]
public sealed partial class RecrdJsonContext : JsonSerializerContext { }
```

Key decisions:
- `GenerationMode = JsonSourceGenerationMode.Metadata` -- REQUIRED for polymorphic `IStep` serialization. Fast-path (Serialization mode) does NOT support polymorphism.
- `PropertyNamingPolicy = JsonKnownNamingPolicy.CamelCase` -- JSON keys are camelCase (e.g., `schemaVersion`, `actionType`). Per research recommendation.
- `DefaultIgnoreCondition = JsonIgnoreCondition.WhenWritingNull` -- Omits null properties (e.g., `DataVariable` when null).
- `WriteIndented = false` -- Compact output by default.
- All concrete types, interfaces, and collection types explicitly registered for source generation.
- `IReadOnlyList<IStep>`, `IReadOnlyList<Variable>`, `IReadOnlyList<Selector>` registered to handle list properties.
- `Dictionary<string, string>` and `IReadOnlyDictionary<string, string>` registered for Payload fields (Research Pitfall 6).
- Context is `public` (not `internal`) so consuming packages can use it for serialization.
